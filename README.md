# Moto Vitals — Website Files

This folder contains everything you need for the Moto Vitals landing page:

- `index.html` — the landing page
- `privacy.html` — Privacy Policy
- `terms.html` — Terms of Use

## Before you go live

Search each file for these placeholders and replace them with your real details:

- `hello@motovitals.app` → your real contact email
- The Facebook link in the Contact section (currently `href="#"`)
- The "Join early access waitlist" link if you set up a real signup form later

## Deploying for free (pick one)

### Option A — Netlify (easiest, drag-and-drop)
1. Go to https://app.netlify.com/drop
2. Drag this whole folder onto the page
3. Netlify gives you a live URL like `https://motovitals.netlify.app` in seconds
4. Optional: in Site settings → Change site name, to make the URL nicer

### Option B — GitHub Pages
1. Create a new GitHub repository (e.g. `moto-vitals-site`)
2. Upload these 3 files to the repository
3. Go to Settings → Pages → set Source to the `main` branch, root folder
4. Your site will be live at `https://yourusername.github.io/moto-vitals-site`

### Option C — Cloudflare Pages
1. Go to https://pages.cloudflare.com
2. Create a project → Direct Upload
3. Upload this folder
4. You'll get a URL like `https://motovitals.pages.dev`

All three options are free with no credit card required, and all support HTTPS out of the box.

## Using this URL for affiliate applications

Once deployed, use the new URL (not the CloudFront link) when applying to affiliate networks like Involve Asia, Lazada, or AliExpress Affiliate. A clean, branded URL with visible About/Contact/Privacy pages reads as a more established property than a raw CDN link.
